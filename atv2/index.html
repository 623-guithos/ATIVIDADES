Apresentação do projeto

Este projeto foi feito para facilitar a gestão escolar, permitindo o cadastro de alunos, professores, disciplinas, turmas, trabalhos e provas. A interface é simples e intuitiva, proporcionando uma experiência agradável para os usuários.

Problemas Identificados
